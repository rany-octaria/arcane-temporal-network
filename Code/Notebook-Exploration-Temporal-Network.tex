\subsection{Exploratory Data Analysis of the Temporal Direct Network
Data}\label{exploratory-data-analysis-of-the-temporal-direct-network-data}

We are running exploratory data analyses of the French transfer
networks. OUr data comes from the direct transfers of all France
hospitals in 2024, all within a 7-day time frame aggregation within one
year. Therefore, our first analysis would be a descriptive analyses of
the basic network characteristics.
