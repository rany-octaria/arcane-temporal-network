Exploratory Data Analysis of the Temporal Direct Network Data

We are running exploratory data analyses of the French transfer networks. OUr data comes from the direct transfers of all France hospitals in 2024, all within a 7-day time frame aggregation within one year. Therefore, our first analysis would be a descriptive analyses of the basic network characteristics.

Weekly Transfer Window Data

Let’s explore the direct transfers with 7-day (weekly) time window,starting for all year timespan.

df <- weekly %>%
